Atividade 1 – Coding

Projeto simples desenvolvido para fins de aprendizado, com foco na integração de API pública em uma aplicação responsiv.

📌 Descrição

Este projeto possui três telas principais:

Login

Cadastro (integração com API pública de CEP)

Home

Na tela de cadastro é utilizada uma API pública de consulta de CEP, permitindo preencher automaticamente algumas informações de endereço.

A interface do projeto também foi desenvolvida de forma responsiva, adaptando-se a diferentes tamanhos de tela.
🛠 Tecnologias utilizadas

HTML

CSS

JavaScript

API pública de CEP

🚀 Deploy

O projeto está publicado na plataforma Vercel.
